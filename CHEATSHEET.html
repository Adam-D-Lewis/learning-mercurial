Mercurial Cheat Sheet

Getting Started

hg clone <url>Clone a repo
hg initCreate new repo
hg statusShow changed files
hg summaryWorking directory summary

Making Changes

hg add <file>Track a new file
hg remove <file>Untrack and delete
hg forget <file>Untrack but keep file
hg commit -m "msg"Commit changes
hg addremoveTrack new, untrack missing

Viewing History

hg logFull history
hg log -GGraph view
hg log -r .Current changeset
hg diffUncommitted changes
hg diff -r X -r YDiff two revisions
hg diff -c REVWhat one changeset changed
hg annotate <file>Blame per line

Syncing with Remote

hg pushPush to remote
hg push -r REVPush specific changeset
hg pullPull from remote
hg pull -uPull and update
hg incomingPreview incoming changesets
hg outgoingPreview outgoing changesets
hg pathsShow configured remotes
hg shelveStash uncommitted changes
hg unshelveRestore stashed changes

Navigating & Branching

hg updateUpdate to branch tip
hg update -r REVGo to specific revision
hg headsShow branch heads
hg branch <name>Set named branch
hg branchesList branches
hg mergeMerge other head
hg resolve --markMark conflicts resolved
hg rebase -s REV -d DESTRebase REV + descendants
hg rebase -r REV -d DESTRebase single REV

Rewriting History (evolve)

hg amendAmend current changeset
hg amend -m "msg"Amend with new message
hg fold --exact -r X::YSquash changesets into one
hg fold --from -r REVFold from REV to current
hg splitSplit changeset into parts
hg uncommitUndo commit, keep changes
hg prune -r REVDiscard a changeset
hg evolveResolve unstable changesets
Draft only. Demote first: hg phase --draft -f -r REV

Phases

hg phase -r REVShow phase
hg phase --public -r REVMark public (immutable)
hg phase --draft -f -r REVDemote to draft (mutable)
hg phase --secret -r REVMark secret (never pushed)

Revision Syntax

.Current changeset
tipLatest changeset
.~1 / .~2Parent / grandparent
X::YX to Y (DAG range)
draft()All draft changesets
obsolete()All obsolete changesets
